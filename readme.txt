== Installation ==
= Minimum Requirements =
* WooCommerce 3.0 or later
* WordPress 4.0 or later

= Automatic installation =
1. Log in to your WordPress dashboard
2. Navigate to the Plugins menu and click Add New.
3. Search and locate 'WooCommerce MultiStep Checkout' plugin.
4. Click 'Install Now'

== Frequently Asked Questions ==

= Can I override the woocommerce template of plugins to my desire? =

Yes, you can. You have to follow same directory structure as the plugins has. 
e.g. If you wish to update template of checkout form, just copy the file 'form-checkout.php' from plugin "w3nuts-wc-multistep-checkout/woocommerce/checkout" and create a same directory structure "w3nuts-wc-multistep-checkout/woocommerce/checkout" into your theme and paste the file. Now, you can edit that file to your need.
